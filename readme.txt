Ревью выполнили: 

## Описание
- Упрощенный аналог du. Добавлены прогрессбар, опции для сопртировки и группировки

## Требования
- Python версии не ниже 3.6 (используется 3.9)

## Состав
- Консольная версия: 'diskusage.py'

## Синтаксис запуска:
- diskusage.py [параметры] [путь-к-элементу-файловой-системы]

## Параметры:
(*) "-a", "--all" - выводить размер для всех файлов, а не только для директорий, по умолчанию размер выводится только для папок
(*) "-r", "--human-readable" - выводить размер в единицах измерения удобных для человека
(*) "-s", "--summarize" - выводить только общий размер
(*) "-S", "--separate-dirs" - не включать размер подпапок в размер папки
(*) "-ads", "--alternate-data-streams" - включить поддержу ADS в NTFS
(*) "-q", "--sort" - выполнить сортировку вывода:
	При -q=0 - нет сортировки
	При -q=1 - сортировка по размеру файла/каталога
	При -q=2 - лексикографическая сортировка по путю файла/каталога
	При -q=3 - сортировка по времени последнего изменения файла/каталога
	При -q=4 - сортировка по времени создания файла/каталога
(*) "-ge", "--group-extension" - группировка по расширению (string)
    "-go", "--group-owner" - группировка по владельцу (string)
    "-gn", "--group-nesting" - группировка по уровню вложенности (int)
    "-gdc", "--group-date-create" - группировка по дате создания (string формата %d.%m.%Y)
    "-gtc", "--group-time-create" - группировка по веремени создания (sting формата %H.%M)
    "-gdl", "--group-date-last-change" - группировка по дате последнего изменения (sting формата %d.%m.%Y)
    "-gtl", "--group-time-last-change" - группировка по времени последнего изменения (sting формата %H.%M)
